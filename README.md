This is an n8n community node for [JetAPI](https://jetapi.io) - send messages via Telegram and WhatsApp.
```bash
npm install n8n-nodes-jetapi
Usage

Get your API credentials from JetAPI Dashboard
Create JetAPI credentials in n8n with your Bearer Token
Add JetAPI node to your workflow
Configure the message parameters

Features

WhatsApp messaging - Send messages via phone number
Telegram messaging - Send via phone number, username, or user ID
Message scheduling - Send messages at specific times
Delivery tracking - Track message status with callbacks
